Code Camp

Joint program to provide training and internship

Overview

You are from Non-CS major, or you don't have any programming experience, or you are just a curious student. This is the program for you.

If you love to learn new technologies, eager to know industry projects and excited about working with a group of technical experts to deliver customer-facing solutions that can help us to do business better, you are encouraged to speak with us. Donot worry about your background, the training in this program will cover it.

Syllabus

SECTION I

Course Introduction

Covers overview of the training course, including assignment, quiz, presentations and marketing procedures etc

SECTION II

Web Basics

Covers essentails elements in web environment, including HTML, CSS, Bootstrap etc. Introduction to NodeJS also included.

SECTION III

Angular

Dedicated section to angular framework. Including components, forms, routes, Http, Observables and so on

SECTION IV

React

Dedicated section to React. Including React Natives, Redux, Routes and so on

SECTION V

Core Java

Most important topics in Java language, such as, thread, concurrency, design pattern, collections, stream API, lambda, OOP design etc

SECTION VI

Database

In-depth tutorial including both SQL and NoSQL. Lots of examples and practice questions covering Oracle, MySQL, SQL Server, MongoDB, Cassandra

SECTION VII

Spring Framework

Full coverage of key spring frameworks. Helping understand how the framework works internally. Covers MVC, Rest API, IoC, AOP etc

SECTION VIII

Spring Boot

Spring boot as a standalone topic to emphasis the importance and usage of this tool. Code walk through with real work examples

SECTION IX

Data Access

A cruise into the common frameworks in DAO layer. Vivid explanation on how industry handle this tough layer. Frameworks include JPA, Hibernate, JdbcTemplate and so on

SECTION X

Industry Project Environment

Simulate project development environment, covering Agile methodologies, CICD tools, Cloud Platforms, DevOps, Documentations and work culture etc

Project

PROJECT A

Mensa

A rewrite of the legacy mensa project. We are enhance our customer facing project with new technologies including Spring Boot, Security, MongoDB, Angular, Bootstrap. The legacy project is featured with MySql, Thymeleaf and Spring framework.

PROJECT L

Virtual-Hub

The virtual-hub project is building a platform to automate the current technical recruiting and marketing process. It uses React, Spring Boot and Cloud technologies. The project will be expanded to cover our other internal projects

PROJECT I

Chamaeleon

Our customer is requesting a system for car dealers. This is a brand new platform to integrate all car dealer products together such that customer can select used and new cars, book appointments in one place
