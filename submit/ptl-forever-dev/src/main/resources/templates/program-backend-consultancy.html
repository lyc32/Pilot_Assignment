Syllabus

SECTION I

Course Introduction

Covers overview of the training course, including assignment, quiz, presentations and marketing procedures etc

SECTION II

Core Java

Most important topics in Java language, such as, thread, concurrency, design pattern, collections, stream API, lambda, OOP design etc

SECTION III

Database

In-depth tutorial including both SQL and NoSQL. Lots of examples and practice questions covering Oracle, MySQL, SQL Server, MongoDB, Cassandra

SECTION IV

Spring Framework

Full coverage of key spring frameworks. Helping understand how the framework works internally. Covers MVC, Rest API, IoC, AOP etc

SECTION V

Spring Boot

Spring boot as a standalone topic to emphasis the importance and usage of this tool. Code walk through with real work examples

SECTION VI

Data Access

A cruise into the common frameworks in DAO layer. Vivid explanation on how industry handle this tough layer. Frameworks include JPA, Hibernate, JdbcTemplate and so on

SECTION VII

Web Security

Join us to feel how our trainer explains web security, you will never feel so clear about how security works. Topic covers include Spring Security, JWT token, OAuth 2, Private and Public Key etc

SECTION VIII

Test Framework

Test in different environment (Local, Dev, QA, Performance, Prod). Multiple test types such as Automation Test, Load Test, AB test etc. Technical coverage includes JUnit, Mockito, Selenium, Cucumber, TestNG and more

SECTION IX

MicroService

We will cover all aspects of microservice ecosystem, including Eureka, Load Balancer, API Gateway, Kafka, RabbitMQ, Feign etc. You will have chance to write your own microservice project

SECTION X

Industry Project Environment

Simulate project development environment, covering Agile methodologies, CICD tools, Cloud Platforms, DevOps, Documentations and work culture etc

Project

Profile Builder

This is an front end demanded project to guide our employees to build their personal profile webpage. This is an entry level project. It helps you utilize the knowledges you learnt and be ready with your own profile when you are presenting yourself

PROJECT E

Ecommerce MicroService

Ecommerce microservice is an training project that integrates spring boot, Rest API, Dependency Injection with Netflix MicroService OSS. Traditional Database and Kafka is also used. A lot of more details will be included in class

PROJECT W

User/Inventory Management System

A backend and full stack system to cover all important topics that industry currently uses. You will build the whole system from front end, backend to database. You will also integrate securities and test frameworks.
